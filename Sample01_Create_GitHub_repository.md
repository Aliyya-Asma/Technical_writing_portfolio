# How to Create a GitHub Repository

## Introduction
This guide explains how to create a repository in GitHub.

## Prerequisites
You must have a GitHub account before creating a repository.

## Steps to Create a Repository

### Step 1: Sign in to GitHub
Go to https://github.com . Click on **Sign in** on top right corner. Enter your email address and password, and then click on Sign in.

### Step 2: Open the New Repository Page
Click on **+** icon in top right corner. Click on **New repository**

### Step 3: Enter Repository Details
Enter an available Repository name and description if required. 

### Step 4: Create the Repository
Select the required configuration options and click on **Create repository**

## Expected Result
You will be taken to the created repository's homepage.

## Troubleshooting

### Unable to Sign in
If the entered Password is incorrect click on **Forgot Password** to set new Password.

## Conclusion
You have successfully created a GitHub repository.
