# How to Install Visual Studio Code on Windows

## Introduction
This guide explains how to install Visual Studio Code on Windows.

## Prerequisites
- A windows computer
- Stable internet connection

## Install Visual Studio Code

### Step 1: Download the Installer
Go to https://code.visualstudio.com . Click on **Download for Windows** to download the Visual Studio Code installer.

### Step 2: Run the Installer
Double click on the downloaded Visual Studio Code installer.

### Step 3: Accept the License Agreement
Accept the agreement and click on **Next**

### Step 4: Select Installation Options
Select the destination location and click on **Next**. And then select the shortcut folder and click on **Next**.
Select other installation options and click on **Next** when finished.

### Step 5: Complete the Installation
Click on **Install** to begin installation of Visual Studio Code.

## Verify the Installation
Open Visual Studio Code from Start menu. If the application launchs succesfully, the installation is complete.

## Troubleshooting

###Installer Could Not Be Downloaded
Check the internet connection and retry the download.

## Conclusion
You have successfully installed Visual Studio Code on Windows.
