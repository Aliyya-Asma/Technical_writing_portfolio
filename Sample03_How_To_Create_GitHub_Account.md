# How to Create a GitHub Account

## Introduction
This guide explains how to create a GitHub account.

## Prerequisites
- A valid email Address
- A stable internet connection

## Create a GitHub Account

### Step 1: Open the GitHub Sign Up Page
Go to https://github.com . Click on **Sign up** in top right corner.

### Step 2: Enter Your Email Address
Enter your email address, create a password, choose your username and select your Country/Region. 

### Step 3: Complete the Verification Process
Complete the automatic verification. A launch code will be sent to your email address. Enter the code and click on **Continue**.

### Step 4: Verify the Account
After the account is created successfully, sign in using your username and password.

## Troubleshooting

### Email Address Is Already Registered
You can sign in with your registered Email address. Click on **Forgot password?** if you don't remember your password. Enter your Email address and click on **Send password reset email**. 
A password reset link will be sent to the email address.

### Username Is Not Available
Set a new username and check for availability. You can also suggest one of the suggested usernames.

### Verification Code Not Received
If you don't receive the verification code, you can click on **Resend the code** or **update your email address** and complete the verification.

## Conclusion
You have successfully created a GitHub account.
